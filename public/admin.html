Start & Online stellen

  1. Im Ordner der App starten: npm start
  2. Im Browser oeffnen: http://localhost:3000
  3. Admin-Passwort setzen: $env:ADMIN_PASSWORD="dein-passwort"; npm start
  4. Online stellen: bei einem Node-Hoster wie Render, Railway oder einem kleinen VPS hochladen und npm start als Startbefehl nutzen.
